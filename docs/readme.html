Typography Theme

Typography is a child theme for Genesis Framework. This theme is designed for proven typography and engagement. To install this plugin you will need to have Genesis Framework. In case you don't have the framework, you can buy Genesis Framework from here. If you don't want to buy the framework then you might like our standalone Typography theme. The standalone version will be available soon.

Aim of the theme

The primary focus of the theme is to provide fastest possible Genesis Theme with most optimised Sharebar, converting Ad units and better call to actions like newsletter sign-ups. If you have any suggestion, please write to anand [AT] blogsynthesis.com or WPAvenue Forums.

Features

Typography is built for the rock-solid Genesis Framework, which is fast, secure and comes with lifetime free upgrade and support. Here are the few highlights of the theme

  • Based on rock-solid Genesis Framework
  • Proven converting homepage
  • Ads ready. Fits the best Adsense units.
  • Default Compressed CSS and script files for faster front-end experience. Also improves Google PageSpeed score
  • Responsive Design
  • Slick and elegant design.
  • Post thumbnails that work with WordPress' featured image functionality and provide extra features.
  • Dynamic body, post, and comment CSS classes.
  • Fully internationalized, so you can run the theme in any language.
  • Floating Social Sharebar

Adding/Removing Features:

Add Logo Image: Upload your logo of 320x120px (and 640x240px for retina display) to /wp-themes/typography/images/.

Add Social Media Buttons: You should install "Simple Social Icons" plugin for this functionality. The default color code is #2aa4cf. Also, Don't forget to replace twitter handles in sharebar.php. By default we have added our own handle

Add email subscription widget: To add the "Email Subscription" widget on the home page. Add enews Plugin.

Remove Share Bar: To remove the floating sharebar remove the following lines from functions.php file

add_action( 'wp_enqueue_scripts', 'custom_enqueue_script' );
function custom_enqueue_script() {
	wp_enqueue_script( 'custom-script', get_stylesheet_directory_uri() . '/js/custom.js', array( 'jquery' ), '', true );
}
